# Documentation Screenshots

This folder contains screenshots and images for the project documentation.

## Required Screenshots

To complete the documentation, add the following screenshots:

1. **screenshot.png** - Main dashboard overview
2. **dashboard-main.png** - Full dashboard with multiple machines
3. **machine-details.png** - Detailed view of a single machine
4. **charts.png** - Historical data charts

## How to Add Screenshots

1. Run the application:
   ```bash
   cd backend
   python api.py
   ```

2. Open browser to `http://localhost:5000`

3. Take screenshots using:
   - Windows: `Win + Shift + S`
   - Mac: `Cmd + Shift + 4`
   - Linux: `Print Screen` or screenshot tool

4. Save screenshots to this folder with the names listed above

5. Recommended screenshot dimensions: 1920x1080 or 1280x720

## Screenshot Guidelines

- Use full browser window
- Show realistic data (let machines run for a few seconds)
- Ensure good lighting/contrast
- Include the green theme clearly
- Show various machine states (RUNNING, IDLE, ALARM)
